# Global Retail Data Engineering

## Project Overview

End-to-end data pipeline for retail analytics built on Databricks

## Tech Stack

- **Platform**: Databricks (DBR 13.3 LTS)
- **Processing**: PySpark, Delta Lake
- **Storage**: DBFS / Delta Tables
- **Orchestration**: Databricks Workflows
